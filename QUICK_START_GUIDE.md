# Quick Start Guide - New Features

## For Customers

### 1. View Your Booking Statistics
- Go to **Dashboard** after logging in
- See your total bookings, spending, and upcoming trips
- Statistics update automatically

### 2. Upload Payment Proof
1. Go to **My Bookings**
2. Find your pending booking
3. Click **Choose File** under "Upload Payment Proof"
4. Select your payment screenshot (GCash, PayMaya, etc.)
5. Preview appears automatically
6. Enter reference number (optional)
7. Click **Upload Payment Proof**
8. Wait for admin verification

**Tips:**
- Maximum file size: 5MB
- Supported formats: JPG, PNG, GIF
- Clear, readable screenshots work best
- Include transaction details in the image

### 3. Cancel a Booking
1. Go to **My Bookings**
2. Find the booking you want to cancel
3. Click **Cancel Booking** button
4. Confirm the cancellation
5. Booking status changes to "Cancelled"

**Note:** Only pending bookings can be cancelled. Contact support for confirmed bookings.

### 4. Track Booking Status
- **Pending** - Waiting for payment
- **Pending Verification** - Payment proof uploaded, awaiting admin review
- **Confirmed** - Booking confirmed by admin
- **Completed** - Trip completed
- **Cancelled** - Booking cancelled

## For Administrators

### 1. View Analytics Dashboard
1. Go to **Admin Dashboard**
2. Click **Reports** in the navigation
3. View comprehensive analytics:
   - Revenue by payment method
   - Booking status distribution
   - Popular activities
   - Top customers

### 2. Export Data
1. Go to **Admin Reports**
2. Choose export type:
   - **Bookings**: Select date range and status filter
   - **Users**: Export all user data
3. Click **Download CSV**
4. Open in Excel or Google Sheets

### 3. Verify Payment Proofs
1. Go to **Admin Bookings**
2. Find bookings with "Pending Verification" status
3. Click on payment proof thumbnail to view full size
4. Verify the payment details
5. Click **Confirm Booking** if valid
6. Click **Cancel Booking** if invalid

### 4. Manage Users
1. Go to **Admin Users**
2. View all registered users
3. Change user roles (Guest, Staff, Admin)
4. Activate/Deactivate user accounts
5. View user statistics

### 5. Monitor System Activity
- All user actions are logged automatically
- Track booking creations, cancellations, and updates
- Monitor payment uploads
- Review user activity patterns

## Common Tasks

### Customer: Make a New Booking
1. Dashboard → **New Booking**
2. Choose booking type:
   - Island Tours
   - Floating Cottage
   - Water Activities
3. Fill in details
4. Select payment method
5. Submit booking
6. Upload payment proof (if not cash)

### Admin: Confirm a Booking
1. Admin Bookings → Find booking
2. Review booking details
3. Check payment proof (if uploaded)
4. Click **Confirm Booking**
5. Customer receives confirmation

### Admin: Generate Monthly Report
1. Admin Reports → Export Bookings
2. Set start date (e.g., 2025-01-01)
3. Set end date (e.g., 2025-01-31)
4. Select status: "All Statuses"
5. Click **Download Bookings CSV**
6. Analyze in spreadsheet software

## Troubleshooting

### Payment Proof Upload Issues
**Problem:** File won't upload
- Check file size (must be under 5MB)
- Ensure it's an image file (JPG, PNG, GIF)
- Try compressing the image
- Check internet connection

**Problem:** Preview not showing
- Refresh the page
- Try a different browser
- Clear browser cache

### Booking Cancellation Issues
**Problem:** Can't cancel booking
- Only pending bookings can be cancelled
- Confirmed bookings require admin contact
- Check booking status first

### Analytics Not Loading
**Problem:** Charts show "Loading..."
- Check internet connection
- Refresh the page
- Ensure you're logged in as admin
- Check browser console for errors
